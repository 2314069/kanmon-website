# 感門之盟 ウェブサイト

## プロジェクト概要
イシス編集学校の講座修了式「第90回 感門之盟」のイベントサイト。

## 開発ルール

### 対象バージョン
- **v6のみを編集対象とする**（v5は変更しない）
- 作業ディレクトリ: `src/v6/`

### ファイル構成
```
src/v6/
├── index.html          # トップページ
├── program.html        # プログラム
├── classroom.html      # 教室一覧
├── info.html           # 参加案内
├── edist.html          # エディスト記事
├── interbooking.html   # インターブッキング
├── apply.html          # 申し込み
├── css/
│   ├── styles.css      # メインCSS（@import で各モジュールを読み込み）
│   ├── base/           # 変数定義
│   ├── components/     # UIコンポーネント
│   ├── sections/       # セクション別スタイル
│   ├── pages/          # ページ固有スタイル
│   └── utilities/      # ユーティリティ
└── js/
    └── main.js         # メインJS
```

### CSS設計
- モジュール化されたCSS構成
- 新しいセクション追加時は `css/sections/_[name].css` を作成し、`styles.css` にインポートを追加
- CSS変数は `css/base/_variables.css` で定義
