## 3. ページ構成

### 3.1 サイト構造
```
トップページ（1ページ構成）
├── ヘッダー
├── メインビジュアル
├── イベント概要
├── タイムテーブル
├── 登壇者紹介
├── 参加案内
├── アクセス
├── ギャラリー
└── フッター
```

### 3.2 各セクション詳細

#### ヘッダー
- ロゴ（感門之盟）
- ナビゲーションメニュー
  - 各セクションへのアンカーリンク
- モバイル時：ハンバーガーメニュー

#### メインビジュアル
- イベントのキービジュアル画像
- イベント名
- 開催日時（目立つ表示）

#### イベント概要
- 開催日時
- 会場名
- 会場住所
- イベントの簡単な説明

#### タイムテーブル
- 当日のプログラム一覧
- 時間 / 内容 / 登壇者 の表形式
- 例：
  | 時間 | 内容 | 登壇者 |
  |------|------|--------|
  | 13:00 | 開場・受付 | - |
  | 13:30 | 開会挨拶 | ○○ |
  | 14:00 | 修了式 | - |
  | ... | ... | ... |

#### 登壇者紹介
- 登壇者ごとのカード表示
  - 写真
  - 名前
  - 肩書き / 所属
  - 紹介文

#### 参加案内
- 受付方法
  - 受付場所
  - 受付時間
  - 受付時に必要なもの
- 持ち物
- 服装（ドレスコードがあれば）
- 注意事項

#### アクセス
- 会場名・住所
- Google Maps 埋め込み
- 最寄り駅からのアクセス方法
- 駐車場情報（該当する場合）

#### ギャラリー
- 過去開催の写真
- グリッドレイアウトで表示
- クリックで拡大表示（Lightbox）

#### フッター
- 問い合わせ先（メールアドレス or フォームリンク）
- 主催者情報
- コピーライト

## 4. デザイン仕様

### 4.1 デザインの方向性
- **親しみやすい・カジュアル**
- 明るく、楽しい、アットホームな雰囲気
- シンプルで見やすい
- 情報が整理されている
- ベースレイアウト：our-blooms.html を活用

### 4.2 カラーパレット（参考）
| 用途 | カラー |
|------|--------|
| 背景 | #FFFFFF |
| 本文 | #2C2825 |
| 見出し | #171615 |
| アクセント | #FFC800 |
| 補足テキスト | #575757 |

※ デザイン時に調整可能

### 4.3 フォント
- 見出し：ゴシック系（太め）
- 本文：ゴシック系（読みやすさ重視）
- 英字：Outfit または類似フォント

### 4.4 レスポンシブブレークポイント
| デバイス | 幅 |
|----------|-----|
| PC | 901px 以上 |
| タブレット | 601px - 900px |
| スマートフォン | 600px 以下 |
