# 感門之盟 Webサイト仕様書

## 1. 概要

### 1.1 プロジェクト情報
| 項目 | 内容 |
|------|------|
| サイト名 | 感門之盟 |
| 目的 | イベント参加者への情報提供 |
| 対象ユーザー | イベント参加者（一般公開） |
| 主催者 | イシス編集学校 |
| 問い合わせ | 既存フォームURL（リンク設置） |

### 1.2 イベント情報
| 項目 | 内容 |
|------|------|
| イベント名 | 90・91回感門之盟 |
| 開催日時 | 2026年3月20日(金)・21日(土) |
| 開催場所 | 豪徳寺 本楼 |
| イベント種類 | 各コース修了式・伝授式 + 交流会 |
| 開催頻度 | 半年に1回 |
| 参加者規模 | 約100名 |

### 1.3 感門之盟とは
感門之盟（かんもんのめい）は、イシス編集学校で講座修了ごとに学びを共にした者同士が一堂に会する集い。校長をはじめ師範陣も学衆も場を共にして編集を楽しむユニークな祝祭。王羲之の蘭亭の会に由来する。

## 3. ページ構成

### 3.1 サイト構造
```
トップページ（1ページ構成）
├── ヘッダー
├── メインビジュアル
├── イベント概要
├── タイムテーブル
├── 登壇者紹介
├── 参加案内
├── アクセス
├── ギャラリー
└── フッター
```

### 3.2 各セクション詳細

#### ヘッダー
- ロゴ（感門之盟）
- ナビゲーションメニュー
  - 各セクションへのアンカーリンク
- モバイル時：ハンバーガーメニュー

#### メインビジュアル
- イベントのキービジュアル画像
- イベント名
- 開催日時（目立つ表示）

#### イベント概要
- 開催日時
- 会場名
- 会場住所
- イベントの簡単な説明

#### タイムテーブル
- 2日間のプログラム一覧（日別に表示）
- 時間 / 内容 の表形式
- 主なプログラム例：
  - **Day 1（3/20）**：○守卒門式、○破突破式、○離退院式 など
  - **Day 2（3/21）**：○花放伝式、○守冠界式、校長校話、交流会 など
- ※各コースの修了式・伝授式の正式名称：
  - 守：卒門式（修了）、冠界式（入門）
  - 破：突破式（修了）
  - 離：退院式（修了）
  - 花伝所：放伝式（修了）

#### 登壇者紹介
- 登壇者ごとのカード表示
  - 写真
  - 名前
  - 肩書き / 所属
  - 紹介文

#### 参加案内
- 受付方法
  - 受付場所
  - 受付時間
  - 受付時に必要なもの
- 持ち物
- 服装（ドレスコードがあれば）
- 注意事項

#### アクセス
- 会場名・住所
- Google Maps 埋め込み
- 最寄り駅からのアクセス方法
- 駐車場情報（該当する場合）

#### ギャラリー
- 過去開催の写真
- グリッドレイアウトで表示
- クリックで拡大表示（Lightbox）

#### フッター
- 問い合わせ先（メールアドレス or フォームリンク）
- 主催者情報
- コピーライト

- **親しみやすい・カジュアル**
- 明るく、楽しい、アットホームな雰囲気
