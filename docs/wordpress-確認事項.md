## 7. 必要なプラグイン（提案）

以下のプラグインがあると便利です。インストール可否を確認：

| プラグイン名 | 用途 | 必要度 |
|-------------|------|--------|
| Custom Post Type UI | 教室投稿タイプ作成 | ★★★ 必須 |
| Advanced Custom Fields | カスタムフィールド管理 | ★★★ 必須 |
| Yoast SEO（または類似） | SEO対策 | ★★ 推奨 |
| Contact Form 7 等 | 問い合わせフォーム | ★ 任意 |

```
│   ├── course/mamoru/      ← 守の一覧
│   ├── course/ha/          ← 破の一覧
│   └── [各教室の個別ページ]
└── dust/89kanmon_.../       ← 既存のレポート記事（通常投稿）
```
